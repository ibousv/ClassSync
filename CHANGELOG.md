# Changelog

All notable changes to ClassSync will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup with Payload CMS 3.0 and Next.js 15
- User authentication with role-based access control (student, delegate, admin)
- Module management for courses (Java, PHP, C#, etc.)
- Resource library with file uploads and view tracking
- Homework management with due dates and completion tracking
- Real-time chat system with Pusher integration
- Activity feed with filtering capabilities
- Comment system for resources and homework
- Reaction system (like, helpful, question)
- Karma and badge system for user contributions
- Lit Web Components for frontend UI
- PostgreSQL database integration
- Vercel deployment configuration
- Docker Compose for local development
- CI/CD pipeline with GitHub Actions
- Comprehensive documentation (Architecture, API, Deployment, Developer Guide, Component Guide)
- Extension example with complete notifications system
- ESLint and Prettier configuration
- TypeScript strict mode
- Access control for all collections
- Real-time event broadcasting
- Media upload with Vercel Blob support

### Security
- JWT-based authentication
- Role-based authorization
- Field-level permissions
- Owner-based access restrictions
- Input validation and sanitization

### Notes
- This is a pre-release version under active development
- Breaking changes may occur before v1.0.0
- Not recommended for production use yet
